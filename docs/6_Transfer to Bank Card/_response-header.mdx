
- **Request Status**
  - **Variable Name**: `applyStatus`  
  - **Required**: Yes  
  - **Type**: String(16)  
  - **Example**: `SUCCESS`  
  - **Description**:  
    - `SUCCESS` – Application successful  
    - `FAIL` – Application failed  
    - `ERROR` – Exception occurred

- **Return Error Code**
  - **Variable Name**: `code`  
  - **Required**: Yes  
  - **Type**: String(10)  
  - **Example**: `0`  
  - **Description**: Return code

- **Return Message**
  - **Variable Name**: `msg`  
  - **Required**: No  
  - **Type**: String(200)  
  - **Example**: `-`  
  - **Description**: Return message
