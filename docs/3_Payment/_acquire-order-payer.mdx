
- **payerMid** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Acquires order payer's member ID.  
  - Example value: `M280311181370`

- **mobileNumberMask** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Payer mobile number with masking applied.  
  - Example value: `+86-13*******00`

- **deviceId** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Payer device fingerprint.  
  - Example value: `1581493898000`

- **ipAddress** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Payer device IP address.  
  - Example value: `1581493898000`
