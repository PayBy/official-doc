
- **requestTime** <font color='#7d8793'>Timestamp</font> <font color='#f19938'>Required</font>  
  - Time the request was initiated.  
  - Example value: `1581493898000`

- **merchantOrderNo** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Unique identifier for the merchant's order.  
  - Example value: `S10000`

- **orderNo** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Unique identifier for the PayBy transaction.  
  - Example value: `O1000`

- **product** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Name of the product or service.  
  - Example value: `Transfer To Bank`

- **status** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Current status of the order.  

- **paymentInfo** <font color='#7d8793'>TransferToBankPaymentInfo</font>  
  - Detailed information about the payment processing, especially related to fees and timing.  
  - Example value:
    
    ```json
    {
      "arrivalTime": 1585133416000,
      "payerFeeAmount": {
        "amount": 0,
        "currency": "AED"
      }
    }
    ```

  - **payerFeeAmount** <font color='#7d8793'>Money</font>  
    - The fee charged to the sender (payer), including amount and currency.  
  
  - **arrivalTime** <font color='#7d8793'>Timestamp (milliseconds, 3-digit precision)</font>  
    - Timestamp when the funds were expected to arrive or did arrive.  
    - Example value: `1585133416000`

- **amount** <font color='#7d8793'>Money</font> <font color='#f19938'>Required</font>  
  - Transaction amount.  

- **holderName** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - SHA-256 hash of the original name.  

- **Iban** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - SHA-256 hash of the original IBAN.  

- **swiftCode** <font color='#7d8793'>String</font>  
  - SWIFT code of the beneficiary bank.  

- **memo** <font color='#7d8793'>String</font>  
  - Remarks or notes for the payment.  

- **notifyUrl** <font color='#7d8793'>String</font>  
  - URL for backend notifications.  

- **failDes** <font color='#7d8793'>String</font>  
  - Reason for order failure.  

- **beneficiaryAddress** <font color='#7d8793'>String</font>  
  - SHA-256 hash of the original address.  

- **bankReference** <font color='#7d8793'>String</font>  
  - Reference number associated with the bank.  

- **fundoutAmount** <font color='#7d8793'>Money</font>  
  - Amount credited to the beneficiary.  

- **rate** <font color='#7d8793'>BigDecimal</font>  
  - Exchange rate used for the transaction.  

- **networkCode** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Financial network used.  
  - Example value: `LOCAL`

- **countryCode** <font color='#7d8793'>String</font>  
  - Country code of the beneficiary bank.  

- **bankName** <font color='#7d8793'>String</font>  
  - Name of the beneficiary bank.  

- **fedwireCode** <font color='#7d8793'>String</font>  
  - Fedwire account code.  

- **intermediaryBank** <font color='#7d8793'>String</font>  
  - SWIFT code of the intermediary bank.  

- **branchName** <font color='#7d8793'>String</font>  
  - Name of the bank branch.  

- **beneficiaryType** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Type of beneficiary account.  
  - Example value: `IBAN`

- **purposeCode** <font color='#7d8793'>String</font>  
  - Code indicating the purpose of the transaction.  
  - Example value: `COM`
