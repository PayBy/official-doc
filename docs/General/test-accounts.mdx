# Test Resources

The following applications and accounts are provided for testing purposes in the sandbox environment only. These resources are not available in the production environment.

## PayBy Mobile Application

### System Requirements

- **Platform**: Android only
- **Purpose**: Testing scan-to-pay functionality, QR code payment display, and related payment scenarios

### Installation

Download the PayBy application using one of the following methods:

**Direct Download**

- [Download Android APK](https://appdownload.payby.com/app-software-download/PAYBY/Payby_4.2.1_2024_09_12_09_59_uat.apk)
