# Query IBAN holder name

## Overview

This API verifies whether the provided account holder name matches the given IBAN.

## Endpoints

- **UAT**: `https://uat.test2pay.com/sgs/api/transfer/getIbanHolderName`  
- **Production**: `https://api.payby.com/sgs/api/transfer/getIbanHolderName`



## Request

### HTTP Headers

| Field            | Name             | Required | Type       | Example | Description                  |
| ---------------- | ---------------- | -------- | ---------- | ------- | ---------------------------- |
| Content-Language | Content-Language | No       | String(10) | `en`    | Language of the response     |
| Signature        | Sign             | Yes      | String     |         | RSA signature                |
| Merchant ID      | Partner-Id       | Yes      | String(12) |         | Assigned merchant identifier |

### HTTP Body

| Field         | Name        | Required | Type                       | Example       | Description       |
| ------------- | ----------- | -------- | -------------------------- | ------------- | ----------------- |
| Request Time  | requestTime | Yes      | Timestamp (ms)             | 1581493898000 | Epoch time in ms  |
| Business Data | bizContent  | Yes      | `GetIbanHolderNameRequest` | -             | Encrypted payload |

#### `GetIbanHolderNameRequest`

| Field       | Name       | Required | Type        | Example | Description         |
| ----------- | ---------- | -------- | ----------- | ------- | ------------------- |
| Holder Name | holderName | Yes      | String(256) |         | Encrypted full name |
| IBAN        | iban       | Yes      | String(256) |         | Encrypted IBAN      |



## Sample Request

```json
{
  "body": {
    "bizContent": {
      "holderName": "<encrypted_holder_name>",
      "iban": "<encrypted_iban>"
    },
    "requestTime": 1585142880000
  }
}
```



## Response

### HTTP Headers

| Field            | Name             | Required | Type       | Example | Description                  |
| ---------------- | ---------------- | -------- | ---------- | ------- | ---------------------------- |
| Content-Language | Content-Language | No       | String(10) | `en`    | Language of the response     |
| Signature        | Sign             | Yes      | String     |         | RSA signature                |
| Merchant ID      | Partner-Id       | Yes      | String(12) |         | Assigned merchant identifier |

### HTTP Body

| Field  | Name | Required | Type                        | Description       |
| ------ | ---- | -------- | --------------------------- | ----------------- |
| Header | head | Yes      | `ResponseHeader`            | Response metadata |
| Body   | body | No       | `GetIbanHolderNameResponse` | IBAN match result |

#### `GetIbanHolderNameResponse`

| Field            | Name           | Required | Type             | Description                  |
| ---------------- | -------------- | -------- | ---------------- | ---------------------------- |
| IBAN Holder Name | ibanHolderName | Yes      | `IbanHolderName` | Masked name and match result |

#### `IbanHolderName`

| Field        | Name               | Type   | Description                              |
| ------------ | ------------------ | ------ | ---------------------------------------- |
| Masked Name  | holderNameMask     | String | Masked version of the account name       |
| Match Level  | nameMatchingLevel  | String | Match confidence level (e.g., "1")       |
| Match Result | nameMatchingResult | String | `"TRUE"` if matched, otherwise `"FALSE"` |
