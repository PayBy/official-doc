
- **applyStatus** <font color='#7d8793'>String(16)</font> <font color='#f19938'>Required</font>  
  - Request status result.  
  - Example value: `SUCCESS`  
  - Possible values:  
    - `SUCCESS`: Application successful  
    - `FAIL`: Application failed  
    - `ERROR`: Exception occurred

- **code** <font color='#7d8793'>String(10)</font> <font color='#f19938'>Required</font>  
  - Return error code.  
  - Example value: `0`

- **msg** <font color='#7d8793'>String(200)</font>  
  - Return message.
