
- **requestTime** <font color='#7d8793'>Timestamp(3)</font> <font color='#f19938'>Required</font>  
  - Request time passed by the merchant when placing the order.  
  - Example value: `1581493898000`

- **merchantOrderNo** <font color='#7d8793'>String(64)</font> <font color='#f19938'>Required</font>  
  - Merchant's reference number for tracking the request.  
  - Example value: `S10000`

- **orderNo** <font color='#7d8793'>String(32)</font> <font color='#f19938'>Required</font>  
  - PayBy's unique identification number of the order.  
  - Example value: `O1000`

- **product** <font color='#7d8793'>String(200)</font> <font color='#f19938'>Required</font>  
  - Product name of the API method used in the order.  
  - Example value: `Transfer Bank Card`

- **status** <font color='#7d8793'>String</font> <font color='#f19938'>Required</font>  
  - Status of the order.  
  - Possible values: `CREATED`, `SUCCESS`, `FAILURE`, `BANK_FAIL`

- **amount** <font color='#7d8793'>Money</font> <font color='#f19938'>Required</font>  
  - Amount to be transferred.

- **accountHolderType** <font color='#7d8793'>String(16)</font> <font color='#f19938'>Required</font>  
  - Account holder type: `INDIVIDUAL` or `CORPORATE`  
  - Example value: `INDIVIDUAL`

- **firstName** <font color='#7d8793'>String(200)</font>  
  - SHA256 of original text. Required if `accountHolderType` is `INDIVIDUAL`.

- **lastName** <font color='#7d8793'>String(200)</font>  
  - SHA256 of original text. Required if `accountHolderType` is `INDIVIDUAL`.

- **middleName** <font color='#7d8793'>String(200)</font>  
  - SHA256 of original text.

- **companyName** <font color='#7d8793'>String(200)</font>  
  - SHA256 of original text. Required if `accountHolderType` is `CORPORATE`.

- **cardNumber** <font color='#7d8793'>String(64)</font> <font color='#f19938'>Required</font>  
  - SHA256 of original card number.

- **expiryMonth** <font color='#7d8793'>String(2)</font> <font color='#f19938'>Required</font>

  - Example value: `09`

- **expiryYear** <font color='#7d8793'>String(4)</font> <font color='#f19938'>Required</font>  
  - Card expiration year.  
  - Example value: `2027`

- **memo** <font color='#7d8793'>String(128)</font>  
  - Transfer description or note.  
  - Example value: `Bonus`

- **notifyUrl** <font color='#7d8793'>String(200)</font>  
  - URL where the merchant receives notifications.  
  - Example value: `http://yoursite.com/api/notification`

- **failDes** <font color='#7d8793'>String(200)</font>

  - Example value: `-`

- **payerFeeAmount** <font color='#7d8793'>Money</font> <font color='#f19938'>Required</font>

    ```json
    {
      "amount": 0,
      "currency": "AED"
    }
    ```

- **payerFeeMemberId** <font color='#7d8793'>String(32)</font> <font color='#f19938'>Required</font>

- **refundedTime** <font color='#7d8793'>Date</font>  
  - Time of refund.
