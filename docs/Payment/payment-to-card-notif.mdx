# Merchant Payment to Card Success Notification

## Overview

When a merchant initiates a payment to a bank card, PayBy sends the transaction result to the merchant. The merchant must process the notification and respond accordingly. The actual time of fund arrival may vary depending on the settlement time of the receiving bank.

## Notification Parameters

| Field Name                 | Variable Name    | Required | Type             | Example | Description                                             |
| -------------------------- | ---------------- | -------- | ---------------- | ------------- | ------------------------------------------------------- |
| Payment to Bank Card Order | `payToCardOrder` | Yes      | `PayToCardOrder` | -             | Contains details of the payment order to the bank card. |

## Response Parameters

Upon receiving the notification, the merchant should return the following JSON response to indicate successful receipt:

```json
{"response":"SUCCESS"}
```

Any other response indicates an error or failure to process the notification.
